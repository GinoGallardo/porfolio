## ⚙️ Instalación y ejecución local

1. **Clonar el repositorio**:
git clone https://github.com/GinoGallardo/porfolio

2. **Instalar las dependencias**:
npm install

3. **Ejecutar el proyecto en modo desarrollo**:
npm run dev

4. **Compilar para producción**:
npm run build
